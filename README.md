# Commands
|Items|Description|
|-----|-----------|
|`now <filename>`| change the current file name|
|`run`|run the code from current file|
|`view`|view all the variables|
|`clear`|renew the runtime|
|`exit`|exit the REPL|
|`cls`|clear the console|
|any codes|execute the code|
